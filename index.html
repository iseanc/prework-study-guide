  • The head element contains information about the webpage.

CSS

  • CSS applies styling (color, alignment, size, etc) to elements in HTML docs
  • CSS can be added inline (style attributes on specific HTML tags), internal CSS stylesheet (added via 'style' element in 'head' section), or external style sheet (separate .css file linked via 'link' tag in 'head' section.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is a code repository and version control system to allow developers to track changes to and store point-in-time versions of their code.
  • GitHub is a cloud-hosted solution used by developers to store code remotely. The service facilitates off-site storage and also collaboration with other people in developing a software solution.
  • 'git' and GitHub havd a workflow that developers use repeatedly. It is basically create a repo on GitHub, clone the repo to a local machine instance of git, create a "branch" to work in temporarily, "add" changes to be pushed up, "commit" changes to the code branch, then "push" them back to main repo in GitHub, where the developer can the request a review and feed back, and eventually merge accepted changes into the main code branch.
